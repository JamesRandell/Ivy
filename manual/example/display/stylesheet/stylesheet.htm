Stylesheets represent the templates that are usually found in either the site folder or i nthe shared theme directory.

When the template engine looks for a stylesheet to use it looks at in these places (first to last):

  • extension/<extension name>/resource/template/
  • site/<site name>/resource/template/
  • (*) shared/<theme name>/template/

(*) The themes in the shared folder are only concerned with template resources, so miss out the 'resource' folder and go directly to the templates.
